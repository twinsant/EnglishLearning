# 不规则学英语
蚂蚁学习英语的历程：总有些不同的做法！

> 从每天10分钟到15分钟到1小时（听-说）

# 动机

能和更多的人交流学习，能获得更多的赚钱机会

# 取经路

多听多说，无他：

* **玩OmeTV**：每天10分钟和外国人聊天，用元宝AI录音笔复盘，这样每天都有进步
  - 看网络评论说在这里容易遇到变态和种族主义者：就我自己的经历，变态没遇到（如果愿意和一个中年大叔展示变态--那可是超级变态了。。。），估计女性用户会遇到这个；种族主义者也没遇到（也有可能是我听不懂，或者我有时就装作不懂/他们就觉得很无趣）
  - 倒是遇到很多泰国、越南、马来西亚等东南亚的小孩子，我一般说个Hi就直接划走
  - 目前为止遇到过巴西的中年大叔聊了如何学外语，英格兰的中年小哥聊了西方的虚假宣传，以色列的小伙子聊双方的情况，和罗马尼亚的姑娘建议英语口语…
  - ⚠️如果遇到对方要你的社交媒体账号，请装傻充愣，毕竟陌生人环境下要有一定的安全意识
* 听Youtube/[HiSpeaking/](https://hispeaking.com/)/[巴别英语](https://www.babelabc.com/)：需要找到语音与大脑关联的模式

## 听力

* 重复多次
  - 隐藏单词，标记听不清的单词
  - 生词本

参考[语音塑造](https://1000h.org/sounds-of-american-english/0-intro.html)，还有[查词典与朗读一样是提高阅读理解能力的最直接有效的手段之一](https://github.com/ZuodaoTech/everyone-can-use-english/blob/main/book/chapter5.md)
